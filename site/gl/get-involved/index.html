Software de escala humana

BookWyrm foi deseñado para ser pequeno e persoal; se queres pedir algunha nova característica, informar de fallos, ou colaborar, encantaríanos que te unises á conversa.

Traballando en BookWyrm

BookWyrm está mantido por Mouse Reeve - que guía o proxecto, xestiona a comunidade de desenvolvedoras, e intenta arranxar os fallos antes de que se presenten. O obxectivo principal de BookWyrm é facelo amigable e acolledor, tanto para a comunidade de desenvolvedoras como software para as usuarias.

Fallos e Solicitudes de características

O xestor de fallos e código de BookWyrm está en GitHub — é o mellor lugar para informar dos fallos e pedir novas funcións e ferramentas.

Traducións

Help translate BookWyrm into other languages! You can learn more about the translation process in our documentation.

Redes sociais

Podes contactar con BookWyrm en Mastodon e Twitter para estar ao día das novas funcións. Etiquétanos para citarnos ou envíanos unha mensaxe directa se queres contactar.

Chat de Desenvolvemento

Usamos Matrix para o chat de desenvolvemento de BookWyrm — é un magnífico lugar para facer preguntas e obter axuda. Está centrado no desenvolvemento do software de BookWyrm e a xestión de instancias, pero é ben vida calquera conversa respecto de BookWyrm.

BookWyrm é software colaborativo e anti-corporacións mantido por Mouse Reeve.
